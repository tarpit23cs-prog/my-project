# 🧸 Toy Shop Management System

A full-stack Toy Shop Management System built using the MERN stack.
This project implements a complete e-commerce flow with role-based access for customers and admin.

---

## 🌐 Live Demo

🔗 Frontend: https://my-project-1-b3yk.onrender.com/
🔗 Backend API: https://my-project-6n8y.onrender.com/

---

## 🚀 Features

### 👤 Customer

* User Signup & Login
* Browse products
* Add to cart
* View & manage cart
* Checkout (Cash on Delivery)
* View orders
* Cancel orders
* View invoice
* Update profile

---

### 🛠️ Admin

* Add new products
* Update existing products
* Delete products
* Manage stock
* View all customer orders

---

## 🧠 Tech Stack

**Frontend**

* React.js
* Bootstrap
* Font Awesome

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas

## ⚙️ Installation & Setup

### Clone Repository

```
git clone https://github.com/tarpit23cs-prog/my-project.git
cd my-project
```

---

### Backend Setup

```
cd toy-backend
npm install
node server.js
```

---

### Frontend Setup

```
cd toyshop-frontend
npm install
npm run dev
```

---

## 🔐 Authentication & Roles

* Role-based access (customer/admin)
* Admin-only access for product & order management

---

## 🛒 Order Flow

1. Add to cart
2. Checkout (COD)
3. Order placed
4. Stock updated
5. Cancel order (optional)
